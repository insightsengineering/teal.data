# List containing default joining keys for `CDISC` datasets

These represent primary and foreign keys for joining `CDISC` datasets.

## Usage

``` r
default_cdisc_join_keys
```

## Format

An object of class `join_keys` (inherits from `list`) of length 19.

## Source

internal. See `data-raw/cdisc_datasets.yaml` and `data-raw/data.R` for
details.

## See also

[`join_keys()`](https://insightsengineering.github.io/teal.data/reference/join_keys.md)
