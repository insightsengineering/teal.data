# Simulated `CDISC` Data for Examples

This package contains random datasets that are used for testing purposes
in the `teal.data` package:

- `rADSL` Random subject-level analysis dataset

&nbsp;

- `rADAE`: Random adverse events analysis dataset

&nbsp;

- `rADCM`: Random concomitant medications analysis dataset

&nbsp;

- `rADEX`: Random response analysis dataset

&nbsp;

- `rADLB`: Random laboratory data analysis dataset

&nbsp;

- `rADRS`: Random response analysis dataset

&nbsp;

- `rADTR`: Random tumor response analysis dataset

&nbsp;

- `rADTTE`: Random time to event analysis dataset

&nbsp;

- `rADVS`: Random vital signs analysis dataset

## Usage

``` r
rADSL

rADAE

rADCM

rADEX

rADLB

rADRS

rADTR

rADTTE

rADVS
```

## Format

- `rADSL`: An object of class `tbl_df` (inherits from `tbl`,
  `data.frame`) with 400 rows and 55 columns.

&nbsp;

- `rADAE`: An object of class `tbl_df` (inherits from `tbl`,
  `data.frame`) with 1934 rows and 92 columns.

&nbsp;

- `rADCM`: An object of class `tbl_df` (inherits from `tbl`,
  `data.frame`) with 3685 rows and 83 columns.

&nbsp;

- `rADEX`: An object of class `tbl_df` (inherits from `tbl`,
  `data.frame`) with 6400 rows and 79 columns.

&nbsp;

- `rADLB`: An object of class `tbl_df` (inherits from `tbl`,
  `data.frame`) with 8400 rows and 102 columns.

&nbsp;

- `rADRS`: An object of class `tbl_df` (inherits from `tbl`,
  `data.frame`) with 3200 rows and 65 columns.

&nbsp;

- `rADTR`: An object of class `tbl_df` (inherits from `tbl`,
  `data.frame`) with 2800 rows and 76 columns.

&nbsp;

- `rADTTE`: An object of class `tbl_df` (inherits from `tbl`,
  `data.frame`) with 2000 rows and 67 columns.

&nbsp;

- `rADVS`: An object of class `tbl_df` (inherits from `tbl`,
  `data.frame`) with 16800 rows and 87 columns.
